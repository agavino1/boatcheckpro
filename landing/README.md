# Landing Page - BoatCheckPro

Este directorio contiene los ficheros para la landing page del proyecto.

## Estructura

- `index.html`: El fichero HTML principal con toda la estructura de la página.
- `style.css`: La hoja de estilos CSS para dar un diseño básico y profesional a la página.

## Próximos Pasos

1.  Revisar y ajustar el contenido (copywriting) del `index.html`.
2.  Mejorar y ampliar los estilos en `style.css`.
3.  Añadir un formulario de contacto funcional (requerirá backend o un servicio de terceros).
4.  Desplegar en un servicio de hosting.
